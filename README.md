<h1 align="center">fp-pweb-confess-dong</h1>

<p align="center">A web application for anonymously sharing thoughts and confessions, built with a focus on privacy and user experience.</p>

## Features

<ul>
  <li>Anonymous posting of confessions</li>
  <li>User-friendly interface</li>
  <li>Responsive design for mobile and desktop</li>
  <li>Moderation tools to ensure a respectful community</li>
  <li>Search and filter confessions by tags or keywords</li>
  <li>Option to upvote or comment on confessions</li>
  <li>User authentication and authorization for secure access</li>
</ul>

## Installation

<ol>
  <li>Clone the repository: <code>git clone https://github.com/your-username/fp-pweb-confess-dong.git</code></li>
  <li>Install dependencies: <code>npm install</code></li>
  <li>Start the development server: <code>npm start</code> (for testing purposes only: <code>npm run dev</code>)</li>
  <li>Open your browser and navigate to <code>http://localhost:4000</code> to view the application.</li>
</ol>

## Usage

<ul>
  <li>To post a confession, click on the "New Confession" button and fill out the form.</li>
  <li>Use the search bar to find specific confessions by keywords or tags.</li>
  <li>Upvote or comment on confessions to engage with the community.</li>
  <li>Moderators can access the moderation panel to review and manage confessions.</li>
</ul>
